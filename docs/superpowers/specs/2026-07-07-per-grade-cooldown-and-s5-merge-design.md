# 등급별 쿨다운 + S5 제거(연출 S6 이관) — 설계 (2026-07-07)

운영 보완 2건. 대상: `kuji-app/index.html` (+ `sw.js`).

## A. 등급별 쿨다운

- 상품 탭 각 상품 행에 **쿨다운(분) 칸** 추가 — `상위` 등급만 활성(일반은 비활성).
- 저장: `prize.coolMin` (설정 저장 버튼으로 커밋, 초기수량과 동일한 방식).
- **빈 칸 = 설정 탭 기본값**(`cfg.cooldownMin`, 기본 15) 사용. **0 = 그 등급만 끔**.
- 설정 탭 "쿨다운(분)"은 **기본값** 역할로 문구 변경. 잠금 키·폴백(풀 비면 무시 지급)·재고 탭 🔒 표시는 기존 유지.
- 상위·일반 토글 변경 시 행 재렌더(쿨다운 칸 활성/비활성 갱신).

## B′. S5(당첨 연출 화면) 제거 — 연출을 S6으로 이관

배경: S6 결과 카드가 S5와 동일한 골드/퍼플 프레임 카드라 S5는 같은 카드를 두 번 보여주는 화면. 제거로 퍼널 터치 1회 감소.

- **새 플로우**: S4 뽑기! → (0.36초) → **S6 결과 화면이 연출과 함께 등장**.
- S6 진입 시:
  - 결과 카드에 **cardIn 등장 애니메이션**(S5의 팝+회전) 적용.
  - **사운드/이펙트 분기 그대로 이관**: 상위/행운상 → `specialFx()`(번개·색종이·**업로드 스페셜 효과음** `sfx_special`), 일반 → **일반당첨음** `sfx_win`. WebAudio 버퍼 재생(v19 수정) 무변경.
- `scr-reveal` HTML/CSS/`showReveal`/`afterReveal` 제거 → `showResult()`로 대체.
- S4 무변경. S6 레이아웃 무변경(진입 연출만 추가). 30초 무입력 복귀 기존대로(`scr-result` 이미 포함).
- 재고 차감 타이밍·로그·일련번호 무변경.
